Стартовый набор для выполнения проектной работы Место
https://github.com/Kw1liache/mesto-ad/blob/main/README.md
###Запускать dev-сервер для разработки: npm run dev
###Собирать проект для продакшена: npm run build
###Просматривать собранный проект: npm run preview
Для запуска приложения необходимо использовать live-server
![изображение live-server](image.png)
Live-server является расширением VSCode
